Funções:

As funções são subrotinas que deixam o código modular mais legivel fazendo com que o codigo possa ser reotilizado. Uma funhção realiza algo espreceifico que pode ser requisitado a quaquer momento.

Exemplos:

  • Crie uma função que leia 3 valores digitados pelo usuario. E realize amultiplicaçoa dos valores e apresente na tela, A função deve ser com parametro e não precisa ter retorno.

    Resolução

  • Altere o exercico anterior em que haja uma função para ler os valores digitados pelo usuario e uma função para calcular a multiplicaçao e ainda uma ultima função para exibir o resultado.

    Resolução.

  • Crie um script em que o usuario digitara a nota da prova, a nota do trabalho e a nota da lista de exercicios e apresente se ele fopi aprovado, reprovado ou exame. A media para a aprovação é 7. Caso a medio fique entre 4 e 6.9 ele esta de exame. Caso contrario estara reprovado. Utilize funçoes com parametros e retorno. para o calculo da media ponderada considerar: 70% para a prova, 20%para trabalho e 10% para a lista de exercicios.

    Resolução

  • Faça um script que realiza a conversão de graus celcius para farenheit.
