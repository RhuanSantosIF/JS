Formularios:

Os formularios permitem aos usuarios fazer a entrada de dados no documento html, A partir destes daods que o usuario insere scripts em linguagem de programação web podem ter acesso e manipular esses dados. Liguagens Front_END como JS, e back_end como PHP,java, phyton entre outras.

Exemplos:

Criar um formulario para somar dois numeros.


No mesmo Script usando os mesmos valores de n1 e n2, cri novos botoes para multiplicar, dividir(N pode ser por 0), e subtrair.


Crie um novo documento html onde o usuario deve entrar o peso e a altura e o programa deve exibir o IMC. Dependendo da faixa do IMC deve informar se esta abaixo do peso, peso normal, sobrepeso e obesidade morbida. Resolução
