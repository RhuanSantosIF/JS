Aprendendo DOM:

E uma APM que permite manipularmos o HTML com o JS. O documento HTML é visto como document e tera associado a ele vario metodos.

  1. getElementById: Retorna o elemento que possui o ID especificado.
  2. getElementsByClassName: Retorna uma lista de elementos que possuem a classe especificada.
  3. getElementsByTagName: Retorna uma lista de elementos que possuem a tag especificada.
  4. innerHTML: Permite obter ou definir o conteúdo HTML de um elemento.
  5. Mudar Estilo:
  6. classList: Permite adicionar, remover e verificar classes de um elemento.
    • .add - adiciona uma classe ao elemento.
    • .remove - remove uma classe do elemento.
    • .toggle - alterna uma classe do elemento.
  7. createElement: Cria um novo elemento HTML.
  8. querySelector: Retorna o primeiro elemento que corresponde ao seletor CSS especificado.
